# Zettel Template

https://zettelkasten.de/overview/

The main idea is to create notes and make links between them.
This template is a folder/naming structure that I use personally.

## Folders Overview

### ENG

I group notes in folders by content.
ENG is for software engineering; I have more root-level folders in my notes vault.
I always create a nested folders structure and use prefixes in file names.
The folder and naming structure is a bit overcomplicated, but that's a good thing -- it tricks me into thinking harder about how to integrate a new note into the existing system, which can result in new notes and unexpected connections.

### GOALS

This is for making goals, creating plans, and tracking progress.

### INBOX

This one is for quick notes to process later.
I often have ideas at random times, so I just write them down to process later.
I don't usually have my laptop with me, though -- in such cases I leave a message in a chat with myself in a messenger app.

### templates

When I create a new empty note, I press `Command + Shift + I` and Obsidian inserts a template -- it saves time and keeps all notes in the same format.

### index files

I use index files for grouping and navigation -- folder grouping doesn't always work.
Index files are always empty; when I need to find all notes related to an index, I check for the Obsidian backlinks.
